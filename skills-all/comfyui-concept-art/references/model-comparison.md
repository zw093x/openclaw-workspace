# 模型对比

## SDXL vs Flux 选型指南

| 维度 | SDXL | Flux |
|------|------|------|
| 画质 | ★★★★ | ★★★★★ |
| 速度 | ★★★★ | ★★★ |
| 显存需求 | 8GB | 12GB+ |
| 人物质量 | 良好 | 优秀（面部、手指） |
| 提示词理解 | 良好 | 优秀 |
| 风格多样性 | 丰富（社区模型多） | 中等（生态发展中） |
| LoRA 支持 | 极好 | 良好 |
| 推荐用途 | 快速迭代、风格探索 | 高质量成品 |

## 选型决策树

```
需要快速出图？
├── 是 → SDXL
└── 否 → 需要高质量人物？
    ├── 是 → Flux
    └── 否 → 需要特定风格 LoRA？
        ├── 是 → SDXL（LoRA 生态更丰富）
        └── 否 → Flux
```

## 推荐模型

### SDXL
- **通用**：sd_xl_base_1.0
- **写实**：RealVisXL
- **动漫**：AnimagineXL
- **概念设计**：DreamShaperXL

### Flux
- **通用**：flux1-dev
- **快速**：flux1-schnell

## 参数适配

| 参数 | SDXL | Flux |
|------|------|------|
| CFG | 7.0 | 3.5-5.0 |
| Steps | 20-30 | 20-25 |
| Sampler | euler_ancestral | euler |
| Resolution | 1024×1024 | 1024×1024 |
