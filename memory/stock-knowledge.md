### 三、缺失模块补充：板块联动与产业链传导

**船舶产业链:**
```
上游: 钢材 → 中游: 造船（中国船舶/中国动力）→ 下游: 航运（中远海控/招商轮船）
```

**中国动力 vs 中国船舶的产业链差异:**

### 三、仓位管理：凯利公式与实战修正

**凯利公式（Kelly Criterion）:**
```
f* = (bp - q) / b
```
- f* = 最优仓位比例
- b = 赔率（盈利/亏损比）
- p = 胜率
- q = 败率（1-p）

**实例计算（船舶当前）:**
- 假设: 反弹至34概率40%，跌破29概率60%
- 盈利空间: 34-31 = 3元
- 亏损空间: 31-29 = 2元
- 赔率 b = 3/2 = 1.5
- p = 0.4, q = 0.6
- f* = (1.5×0.4 - 0.6) / 1.5 = 0 / 1.5 = **0%**

**含义:** 按凯利公式，当前船舶的期望值为0或负，不应持有任何仓位。

**但实际持有3000股（100%仓位），说明:**
1. 被套后不舍得割肉（处置效应偏差）
2. 仓位管理完全失控
3. **这是最大的教训：进场前算好凯利，被套后纪律止损**

**实战修正版凯利（Half-Kelly）:**
- 学术建议: 永远只用凯利值的50%（Half-Kelly）
- 原因: 凯利公式假设已知真实概率，但实际都是估算
- Half-Kelly降低波动性，代价是略降收益，但大幅降低爆仓风险

**未来建仓规则:**
| 信号强度 | 胜率估算 | 凯利仓位 | Half-Kelly | 单股上限 |
|---------|---------|---------|-----------|---------|
| 强（四维A级） | 60% | 40% | **20%** | 25% |
| 中（四维B级） | 50% | 20% | **10%** | 15% |
| 弱（四维C级） | 40% | 0% | **0%** | 不建仓 |

**当前持仓 vs 规则对比:**
- 船舶C级（51分）→ 按规则不应建仓 → 实际持有100% → **严重违反纪律**
