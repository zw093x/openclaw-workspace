## 一、基本面分析

### 财报三大表
- **利润表**: 营收→毛利→营业利润→净利润，关注增速和利润率变化趋势
- **资产负债表**: 资产结构、负债率、商誉风险、应收账款周转
- **现金流量表**: 经营性现金流 vs 净利润（判断盈利质量），自由现金流

### 核心指标
| 指标 | 含义 | 船舶行业参考值 |
|------|------|---------------|
| PE | 市盈率，估值高低 | 周期股看PB更准 |
| PB | 市净率 | <1可能低估，>3警惕 |
| ROE | 净资产收益率 | >15%优秀，>20%卓越 |
| 毛利率 | 产品竞争力 | 造船业通常10-20% |
| 资产负债率 | 财务风险 | >70%需关注 |
| 经营现金流/净利润 | 盈利质量 | >1为健康 |

### 估值方法
- **DCF**: 折现现金流，适合稳定增长企业
- **相对估值**: PE/PB与同行对比
- **周期股**: PB+景气度位置判断
- **重置成本法**: 船坞/设备等重资产企业适用

### 杜邦分析
ROE = 净利率 × 资产周转率 × 权益乘数
- 高净利率驱动（茅台模式）
- 高周转驱动（沃尔玛模式）
- 高杠杆驱动（银行模式）

## 二、技术面分析

### K线核心形态
- 底部反转：早晨之星、锤子线、看涨吞没
- 顶部反转：黄昏之星、射击之星、看跌吞没
- 持续形态：旗形、楔形、矩形整理

### 均线系统
- **MA5**: 短期趋势，短线生命线
- **MA10**: 波段操作参考
- **MA20**: 中期趋势，月线
- **MA60**: 季线，牛熊分界线
- **MA120/250**: 半年线/年线，长期趋势
- **多头排列**: MA5>MA10>MA20>MA60（强势）
- **空头排列**: 反过来（弱势）

### 量价关系
| 场景 | 含义 | 操作建议 |
|------|------|---------|
| 放量上涨 | 主力进场，趋势确认 | 持有/加仓 |
| 缩量上涨 | 跟风不足，可能见顶 | 警惕 |
| 放量下跌 | 恐慌出逃 | 减仓 |
| 缩量下跌 | 惜售，可能见底 | 观察 |
| 量价背离 | 趋势可能反转 | 注意信号 |

### 技术指标
- **MACD**: 金叉/死叉、顶底背离、零轴上方强势
- **KDJ**: 超买(>80)/超卖(<20)、J值拐点
- **RSI**: 同上，更稳定
- **布林带**: 收口→突破、上轨压力/下轨支撑
- **OBV**: 量能先行指标

### 斐波那契回调位
| 回调位 | 含义 | 实战意义 |
|--------|------|---------|
| 23.6% | 浅回调 | 强势调整 |
| 38.2% | 常见回调 | 正常回调 |
| 50.0% | 中等回调 | 中性 |
| 61.8% | 黄金分割 | 最重要支撑/压力位 |
| 78.6% | 深回调 | 趋势可能反转 |

**61.8%规则**: 大多数趋势回调不超过61.8%，跌破则趋势可能反转

### 艾略特波浪理论（简化）
- 推动浪: 5浪结构（1-2-3-4-5）
- 调整浪: 3浪结构（A-B-C）
- 浪2回调通常为浪1的50%-61.8%
- 浪3通常最长，浪4不与浪1重叠
- **注意**: 主观性强，作为辅助工具使用

## 三、行业研究框架（以船舶为实战）

### 行业分析四维度
1. **需求端**: 新船订单量、手持订单/运力比
2. **供给端**: 船坞产能利用率、交付周期
3. **价格端**: 船价指数（Clarksons Newbuilding Price Index）
4. **成本端**: 钢材价格、人工成本

### 船舶行业周期判断
- **复苏期**: 订单触底回升、船价企稳
- **繁荣期**: 订单爆满、船价飙升、船坞排期到2年后
- **衰退期**: 订单下滑、船价松动
- **萧条期**: 船坞停产、大量拆船

### 关键跟踪指标
- Clarksons新船价格指数
- BDI/CCFI运价指数
- 全球手持订单量
- 船坞产能利用率
- 钢材价格（成本端）
- IMO环保法规推进节奏

## 四、宏观经济学

### 美林时钟（4阶段模型）
| 阶段 | 经济增长 | 通胀 | 最佳资产 | A股偏好 |
|------|---------|------|---------|---------|
| 复苏 | ↑ | → | 股票 | 周期股领涨（有色/船舶/建材） |
| 过热 | ↑ | ↑ | 大宗商品 | 资源股/上游 |
| 滞胀 | → | ↑ | 现金 | 防御板块（医药/公用事业） |
| 衰退 | ↓ | → | 债券 | 成长股/消费 |

**2026年Q1位置**: 弱复苏（PMI~50，CPI温和，GDP目标5%）
→ 周期股有弹性但受限，成长股受益于流动性宽松

### 货币政策传导链
```
降准/降息 → 银行间利率↓ → 贷款利率↓ → 融资成本↓
    → 实体投资↑ + 地产销售↑ → 上下游复苏
    → 流动性宽裕 → 资金入市 → 权益资产上涨
```
**注意**: 降息周期尾声 → 边际效应递减

### 汇率与北向资金
- 人民币贬值 → 北向资金流出 → A股承压（尤其蓝筹）
- 人民币升值 → 外资回流 → 利好权重股
- 美联储降息预期 → 美元走弱 → 利好新兴市场

### 重点关注数据（优先级排序）
1. **PMI**: 制造业50为荣枯线（当前弱复苏，50附近）
2. **LPR**: 贷款市场报价利率，直接反映融资成本
3. **社融/M2**: 流动性指标，社融增速=信用扩张速度
4. **进出口数据**: 贸易景气度，与船舶订单强相关
5. **CPI/PPI**: 判断通胀位置和美林时钟阶段
6. **美联储利率决议**: 全球资金面，影响汇率和北向资金
7. **美元/人民币汇率**: 外资进出的直接驱动

## 六、行为金融学（认知偏差与情绪管理）

### 七大核心偏差
| 偏差 | 定义 | 典型表现 | 克服方法 |
|------|------|---------|---------|
| 锚定效应 | 过度依赖参考点 | "成本38.7不回本不卖" | 关注当前价值而非成本 |
| 损失厌恶 | 亏损痛苦>盈利快乐2倍 | 浮亏不愿止损 | 止损=保险费 |
| 确认偏差 | 只看支持自己观点的信息 | 看多只看利好 | 主动找反面论据 |
| 过度自信 | 高估判断能力 | "缩量=止跌" | 用准确率数据校准 |
| 处置效应 | 早卖盈利股、久持亏损股 | +4.8%就卖，-20%还拿 | 基于信号而非盈亏决策 |
| 羊群效应 | 跟随多数人行动 | 追涨杀跌 | 独立思考、逆向投资 |
| 近期偏差 | 过度关注最近事件 | 跌了就觉得完了 | 拉长时间维度 |

### 市场情绪指标
- **恐慌贪婪指数**: <25极度恐慌(买入机会), >75极度贪婪(卖出信号)
- **VIX恐慌指数**: >30恐慌, <15平静（A股替代: 中国波指iVIX）
- **换手率**: 极低=底部区域, 极高=狂热顶部

### 自我检视清单
- [ ] 是否在用成本价做决策？（锚定效应）
- [ ] 是否只看利好忽略利空？（确认偏差）
- [ ] 是否在等待"回本"？（损失厌恶）
- [ ] 是否因为近期涨跌改变长期判断？（近期偏差）
- [ ] 如果现在空仓，会不会买入？（去锚定思维）

## 七、风险管理

### 仓位管理规则
| 规则 | 建议值 | 当前持仓 | 合规 |
|------|--------|---------|------|
| 单只股票仓位上限 | ≤30% | 船舶45.3% | ❌ |
| 行业集中度 | ≤50% | 船舶91.3% | ❌ |
| 现金比例 | ≥20% | 0% | ❌ |
| 总浮亏上限 | ≤15% | -14.2% | ⚠️ |

### 止损纪律（3层止损）
1. **固定比例止损**: 成本价下方-8%（第一道防线）
2. **技术止损**: 跌破关键均线(MA20)或支撑位
3. **时间止损**: 持有N个交易日无表现则退出

### 风险指标
- **波动率**: 日波动率 = √(Σ(收益率-均值)²/n)，年化 = 日波动率 × √252
- **最大回撤**: (峰值-谷值)/峰值，衡量最坏情况
- **VaR风险价值**: 1.65 × 日波动率 × 持仓市值（95%置信度）

### 当前持仓风险诊断（2026-03-27）
- 船舶行业集中度91.3% → 严重超限
- 中国船舶浮亏-20.7% → 应在-8%止损
- 总浮亏-14.2% → 接近15%红线
- 改进: 以后建仓同时设止损单，行业分散

## 九、A股交易规则速查

### 交易时间
- 集合竞价: 09:15-09:25（09:20后不可撤单）
- 连续竞价: 09:30-11:30, 13:00-15:00
- 尾盘竞价: 14:57-15:00（深市）

### 涨跌停限制
- 主板: ±10% | 创业板/科创板: ±20% | 北交所: ±30% | ST: ±5%

### 交易费用
- 买入: 佣金(万2.5-3) + 过户费(万0.1，沪市)
- 卖出: 佣金 + 印花税(千1) + 过户费
- 保本涨幅: 约0.17%（买卖各一次）

### T+1规则
- 今天买入 → 明天才能卖出
- 应对: 分批建仓，尾盘买入减少隔夜风险
