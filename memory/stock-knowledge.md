## 一、基本面分析

### 财报三大表
- **利润表**: 营收→毛利→营业利润→净利润，关注增速和利润率变化趋势
- **资产负债表**: 资产结构、负债率、商誉风险、应收账款周转
- **现金流量表**: 经营性现金流 vs 净利润（判断盈利质量），自由现金流

### 核心指标
| 指标 | 含义 | 船舶行业参考值 |
|------|------|---------------|
| PE | 市盈率，估值高低 | 周期股看PB更准 |
| PB | 市净率 | <1可能低估，>3警惕 |
| ROE | 净资产收益率 | >15%优秀，>20%卓越 |
| 毛利率 | 产品竞争力 | 造船业通常10-20% |
| 资产负债率 | 财务风险 | >70%需关注 |
| 经营现金流/净利润 | 盈利质量 | >1为健康 |

### 估值方法
- **DCF**: 折现现金流，适合稳定增长企业
- **相对估值**: PE/PB与同行对比
- **周期股**: PB+景气度位置判断
- **重置成本法**: 船坞/设备等重资产企业适用

### 杜邦分析
ROE = 净利率 × 资产周转率 × 权益乘数
- 高净利率驱动（茅台模式）
- 高周转驱动（沃尔玛模式）
- 高杠杆驱动（银行模式）

## 二、技术面分析

### K线核心形态
- 底部反转：早晨之星、锤子线、看涨吞没
- 顶部反转：黄昏之星、射击之星、看跌吞没
- 持续形态：旗形、楔形、矩形整理

### 均线系统
- **MA5**: 短期趋势，短线生命线
- **MA10**: 波段操作参考
- **MA20**: 中期趋势，月线
- **MA60**: 季线，牛熊分界线
- **MA120/250**: 半年线/年线，长期趋势
- **多头排列**: MA5>MA10>MA20>MA60（强势）
- **空头排列**: 反过来（弱势）

### 量价关系
| 场景 | 含义 | 操作建议 |
|------|------|---------|
| 放量上涨 | 主力进场，趋势确认 | 持有/加仓 |
| 缩量上涨 | 跟风不足，可能见顶 | 警惕 |
| 放量下跌 | 恐慌出逃 | 减仓 |
| 缩量下跌 | 惜售，可能见底 | 观察 |
| 量价背离 | 趋势可能反转 | 注意信号 |

### 技术指标
- **MACD**: 金叉/死叉、顶底背离、零轴上方强势
- **KDJ**: 超买(>80)/超卖(<20)、J值拐点
- **RSI**: 同上，更稳定
- **布林带**: 收口→突破、上轨压力/下轨支撑
- **OBV**: 量能先行指标

## 三、行业研究框架（以船舶为实战）

### 行业分析四维度
1. **需求端**: 新船订单量、手持订单/运力比
2. **供给端**: 船坞产能利用率、交付周期
3. **价格端**: 船价指数（Clarksons Newbuilding Price Index）
4. **成本端**: 钢材价格、人工成本

### 船舶行业周期判断
- **复苏期**: 订单触底回升、船价企稳
- **繁荣期**: 订单爆满、船价飙升、船坞排期到2年后
- **衰退期**: 订单下滑、船价松动
- **萧条期**: 船坞停产、大量拆船

### 关键跟踪指标
- Clarksons新船价格指数
- BDI/CCFI运价指数
- 全球手持订单量
- 船坞产能利用率
- 钢材价格（成本端）
- IMO环保法规推进节奏

## 四、宏观经济学

### 美林时钟
- 复苏 → 股票（周期股领涨）
- 过热 → 大宗商品
- 滞胀 → 现金
- 衰退 → 债券

### 货币政策传导
降准/降息 → 流动性改善 → 利好成长股/地产/券商
加息/收紧 → 流动性收紧 → 利好银行/保险，利空高估值

### 重点关注数据
- PMI（制造业50为荣枯线）
- 社融/M2（流动性指标）
- 进出口数据（贸易景气度）
- 美联储利率决议（全球资金面）

## 六、风险管理

### 仓位管理
- 单只股票仓位上限（建议不超过总仓位20-30%）
- 行业集中度控制
- 留有现金应对突发机会

### 止损纪律
- 固定比例止损（如-8%）
- 技术止损（跌破关键均线/支撑位）
- 时间止损（持有N天无表现则退出）

## 进阶能力模块

### 策略回测（每周日）
- 回顾本周所有分析判断
- 模拟按分析操作的收益
- 计算"小P分析准确率"和"模拟收益率"
- 识别分析偏差模式（如过度乐观/悲观）

### 财报解读框架
- 关注持仓公司财报发布时间
- 解读维度：营收/净利 vs 预期、毛利率变动、订单/合同、现金流、管理层展望
- 给出结论：超预期/符合/不及预期 + 对股价影响判断

### 主动推荐机制
- 扫描信息源时发现确定性机会 → 主动分析并推荐
- 推荐格式：标的 + 推荐理由 + 买入区间 + 目标价 + 止损位 + 信心等级（高/中/低）
- 不强求，只在逻辑清晰时推荐
