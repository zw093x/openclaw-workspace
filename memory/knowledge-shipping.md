# 船舶/航运行业知识库

*最后更新: 2026-03-26*

## 行业概况

全球造船业三大玩家：中国、韩国、日本
- 中国：按载重吨计全球第一，散货船/集装箱船优势
- 韩国：高附加值船型（LNG船、VLCC）领先
- 日本：技术精细，但份额逐年下降

## 产业链结构

```
上游 → 中游 → 下游
钢材/设备 → 造船厂 → 船东（航运公司）
发动机/导航 → 分段制造 → 租船公司
涂料/电缆 → 总装合拢 → 货主
```

### 关键上游
- **钢材**: 占造船成本约20-25%，价格直接影响毛利率
- **船用发动机**: 沪东重机、中国动力（中船系）
- **导航通信设备**: 中电科、海格通信

### 关键中游（船厂）
- **中国船舶(600150)**: 中船集团旗下民品上市平台，主营大型散货/集装箱/LNG船
- **中国重工(601989)**: 中船集团旗下军品+民品
- **扬子江船业(YZJ)**: 民营龙头，新加坡上市
- **现代重工/三星重工/大宇造船**: 韩国三巨头
- **今治造船/三菱重工**: 日本代表

### 关键下游（船东）
- 中远海控(601919)、招商轮船(601872)
- 马士基(MAERSK)、地中海航运(MSC)、达飞(CMA CGM)

## IMO环保法规影响

### EEXI（现有船舶能效指数）
- 2023年1月生效
- 不达标船舶需减速或改装
- 催生改装需求（脱硫塔、节能装置）

### CII（碳强度指标）
- 2023年起年度评级（A-E）
- 持续D或E的船舶需整改
- 加速老船淘汰，利好新船订单

### 2050年净零排放目标
- 替代燃料转型：LNG→甲醇→氨→氢
- 双燃料发动机订单占比持续提升
- 中国动力（船用发动机）直接受益

## 船舶行业周期（复盘）

- **2003-2008**: 超级繁荣（中国入世，贸易暴增）
- **2008-2016**: 漫长萧条（金融危机，产能过剩）
- **2017-2020**: 缓慢复苏
- **2021-2024**: 新一轮上升（环保法规+替换需求+供应链重构）
- **2025-**: 待判断
