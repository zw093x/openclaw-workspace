- **代理部署成功**，OpenRouter、httpbin 等外网完全可访问

  - `fonts.googleapis.com`（HTTP）可返回 404，说明 HTTP 层可达，HTTPS/TLS 被拦截

- 代理服务商的订阅链接使用 Cloudflare 企业级防火墙
- 数据中心 IP 被直接封禁（非验证码，是硬拦截）

### 三、技术收获

3. **排障思路**：从简单到复杂（TCP → HTTP → HTTPS → TLS → 不同协议/库）
4. **绕过方案**：OpenRouter 作为 API 中转层，可访问 Google 模型
