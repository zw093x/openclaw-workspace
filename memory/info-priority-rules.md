# 信息优先级过滤规则

*最后更新: 2026-03-26*
*用途: 250+信息源的分级处理，避免信息过载*

---

## 🔴 一级 — 实时推送（立即飞书通知）

| 类型 | 触发条件 | 推送方式 |
|------|---------|---------|
| 持仓异动 | 涨跌幅超±3% | 立即推送 |
| 持仓紧急 | 涨跌幅超±5%、触及止损 | 紧急推送 |
| 减仓触发 | 进入减仓价位区间 | 立即推送+操作建议 |
| 技术突破 | 突破/跌破关键技术位（MA20/支撑/压力） | 立即推送 |
| 黑天鹅 | 重大突发事件（地缘冲突、制裁、自然灾害影响供应链） | 紧急推送 |
| 政策重大变化 | 直接影响持仓行业的政策（船舶补贴、芯片制裁等） | 立即推送 |
| 持仓公司重大公告 | 业绩预告、重大合同、高管变动、资产注入 | 立即推送 |
| 大盘极端波动 | 上证单日跌超2%或涨超2% | 立即推送 |

## ⚪ 四级 — 仅记录不推送

| 类型 | 处理方式 |
|------|---------|

| 一般性数据 | 记录到行业知识库 |

## 过滤规则

### 降噪规则
1. **同一事件多源报道** → 只推送最权威的一个来源
2. **已推送的延续报道** → 除非有重大新进展，否则不重复推送
3. **市场噪音**（小幅震荡、常规交易）→ 不推送
4. **与持仓无关的行业新闻** → 仅记录，不推送
5. **KOL观点** → 必须以权威源印证后才可推送

### 升级规则
1. 某条🟡级新闻如果涉及多个持仓 → 升级为🔴
2. 某条🟢级信息如果持续发酵 → 升级为🟡
3. 盘中发现异常信号 → 自动触发深挖机制（见下）

---

*执行时严格按此分级。宁可少推，不可多推造成干扰。*

## 中国动力/中船动力 行业监控（2026-04-01 新增）

### 必须优先推送的消息类型
1. 中船动力/中国动力发动机新订单（尤其是双燃料）
2. 发动机价格变动或交付周期变化
3. 沪东中华/中国船舶批量采购发动机
4. IMO法规更新（环保标准趋严）
5. 氨/甲醇/LNG双燃料发动机技术突破
6. 船东抢购发动机的报道

### 关键信息源
- 中国船舶集团官网 (cssc.net.cn)
- 国际船舶网 (eworldship.com)
- 信德海事网 (xindemarinenews.com)
- 铸造头条 (zhuzaotoutiao.com)
- 中国船舶报

### 监控频率
- 每日盘中至少检查1次
- 发现重大消息立即推送
