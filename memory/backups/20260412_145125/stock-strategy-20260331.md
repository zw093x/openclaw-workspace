## 决策框架

### 一、减仓位判断依据

| 依据 | 说明 |
|------|------|
| 压力位 | 前期成交密集区/高点回落位 |
| 均线阻力 | 价格反弹到MA20/MA60遇阻 |
| 布林上轨 | 触及或突破布林带上轨，短期超买 |
| 量价背离 | 价格上涨但成交量萎缩，反弹乏力 |
| 大盘压力区 | 上证进入3900-3950，系统性减仓 |

**核心逻辑：** 分批降低暴露，接受可能踏空一部分利润。减仓不是卖在最高点。

### 二、止损位判断依据

| 依据 | 说明 |
|------|------|
| 跌破关键支撑 | 前低/平台下沿/整数关口 |
| 均线破位 | 跌破MA60/MA120，趋势转空 |
| 最大亏损容忍 | 个人风险偏好，如总仓位亏损-25% |
| 基本面恶化 | 行业逻辑破坏（非技术面） |

**核心逻辑：** 止损是保护本金，不是认输。错了就认，留着钱等下次机会。

### 三、加仓位判断依据

| 依据 | 说明 |
|------|------|
| 支撑位企稳 | 跌到前期低点/均线支撑，缩量企稳不再创新低 |
| 底部形态 | 双底/W底/头肩底等形态确认 |
| 放量突破 | 突破压力位且放量，确认趋势反转 |
| 板块共振 | 船舶板块整体走强，不是个股孤立反弹 |
| 大盘配合 | 上证站稳关键位，系统性风险降低 |

**加仓比例：** 分批加，不一次打满。
- 试探性加仓 1/3 → 确认后加 1/3 → 趋势确立加 1/3
- 单只股票加仓不超过总仓位 10%

### 四、止盈位判断依据

| 依据 | 说明 |
|------|------|
| 目标价达成 | 买入时设定的合理估值目标 |
| 阻力密集区 | 上方大量套牢盘，继续上涨压力大 |
| RSI超买 | RSI > 70-80，短期过热 |
| 板块见顶信号 | 船舶板块整体滞涨或龙头补跌 |

### 五、踏空应对

**踏空是计划的一部分，不是失败。**

| 情况 | 应对 |
|------|------|
| 减仓后继续涨 | 少赚了一些，但降低了风险暴露（可接受） |
| 减仓后下跌 | 少亏了很多，还保留了加仓的子弹（有利） |
| 减仓后突破压力位放量 | 不追回去，等回踩确认再小仓位接回 |
| 减仓后回踩原压力位变支撑 | 可以接回部分（如减了500股，接回200股） |

**关键原则：永远不要因为踏空而情绪化追高。**
