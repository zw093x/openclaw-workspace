# 即梦AI角色生成三视图结构化提示词公式

> 来源：豆包帖子 https://www.doubao.com/thread/w63df6e5d8038c028
> 抓取时间：2026-04-07
> 适配：即梦AI全系列模型（Seedream 2.0 Pro 最优解、OmniHuman 数字人模型）

---

## 核心铁则

**高权重前置 + 全维度一致性强锁 + 细节全锚定 + 反向精准避坑**

1. 核心指令必须放在提示词靠前位置（即梦AI权重前高后低）
2. 必须明确「三个视图为同一角色的不同视角，所有细节100%统一」
3. 使用标准术语：正视图、正右侧视图、背视图；禁用「侧面、反面」
4. 布局强制定义：水平等距排列，1:1:1等比例，无重叠、无遮挡、无裁切

---

## 8个公式模块

### 模块1：ID一致性强控（进阶版必填）
- 通用：`三个视图为完全同一角色，100%统一，无任何细节差异`
- 写实人像：锁定参考图人物五官特征、面部轮廓，无五官变形
- 二次元/游戏：同一角色三视图，设定完全统一，无画风跳变

### 模块2：角色核心基础设定（必填）
必填要素：年龄段 + 性别 + 核心身份
示例：20岁古风仙侠女性、25岁现代潮酷男性、16岁日系二次元魔法少女

### 模块3：三视图视角与布局（必填核心）
必填指令：
```
三个视图为同一角色的正面/侧面/背面三个角度；
水平等距排列，1:1:1等比例展示；
无重叠、无遮挡、无裁切；
正视图、正右侧视图、背视图，三个视角均完整呈现
```
- 半身三视图：胸部以上为标准
- 建模专用：正交视角，无透视畸变，适合3D建模参考

### 模块4：身材/五官/发型锁定（权重1.4-1.6）
填写框架：`[身材词]:1.5, [脸型]:1.5, [眼型]:1.4, [鼻型][唇形]:1.4, [肤色]:1.2, [发型]:1.4, [发色]:1.3`

身材：高挑纤细、娇小玲珑、丰满匀称、健硕肌肉型、七头身、八头身、Q版二头身
五官：鹅蛋脸、瓜子脸、丹凤眼、桃花眼、高鼻梁、小翘鼻、M唇、冷白皮、黄白皮
发型：黑长直、大波浪卷发、高马尾、丸子头、双马尾、寸头

### 模块5：服饰/配饰全细节锚定
填写顺序：整体款式 → 分部位细节 → 材质 → 配饰

### 模块6：材质/色彩/光影统一
材质：哑光棉麻、亮面丝绸、磨砂皮革、金属拉丝、真皮质感
光影：均匀平光、正交无影光、柔和漫反射光

### 模块7：风格与画质增效词
- 通用必加：8K超清，4K高清，极致细节，无模糊，无噪点
- 游戏建模：PBR次世代渲染，OC渲染，Blender渲染，正交视角
- 二次元：日系动漫画风，赛璐璐画风，平涂上色，无崩坏
- 写实人像：超写实质感，照片级真实感，无面部变形
- 古风/国风：工笔画质感，国风配色，纹样清晰

### 模块8：专属负面避坑词（必填）
```
畸形五官，模糊面部，多余手指、融合手指，五官不对称，肢体畸形，比例失调
缺视图，单视图，多视图，视图重叠，透视错误
杂乱背景，多余元素，水印，文字，logo，画质低，崩坏
```

## 最优参数

| 参数 | 最优设置 |
|------|---------|
| 模型 | Seedream 2.0 Pro |
| 分辨率 | 横屏16:9，2048×1152 或 2560×1440（禁止竖屏） |
| 生成数量 | 4张（多选） |
| 风格强度 | 6-7（0-10分） |
| 随机种子 | 生成满意后固定 |

---

## 翻车问题解决

| 问题 | 方案 |
|------|------|
| 三视图五官不一致 | 统一指令前置 + ID锁定参考图 + 固定种子 |
| 缺视图/单视图 | 布局规范第二顺位 + 横屏16:9 + 反向词加「缺视图」 |
| 视图重叠 | 等距排列 + 纯白背景 + 正交视角 |
| 比例失衡/畸形 | 标注头身比 + 反向词加「畸形、穿模」 |

## 进阶技巧

1. **参考图辅助**：上传设定图，开启风格模仿+内容参考，强度4-5
2. **局部重绘**：单个视图有误，智能画布局部修正
3. **分阶段生成**：低分辨率初稿→筛选→固定种子→高分辨率成品
