## 飞书 streaming 配置

- **问题**: 飞书消息重复显示（文本消息 + 交互卡片各显示一次）
- **原因**: `channels.feishu.streaming` 为 true，导致 OpenClaw 同时发送文本消息和流式交互卡片
- **修复**: 在 openclaw.json 中添加 `"streaming": false` 到 `channels.feishu`
- **注意**: 需要重启 gateway 才能生效
