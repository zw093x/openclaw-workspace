## 一、正负向双向控制

### 核心原理
即梦AI遵循 **"精准指令 = 正向描述 + 反向约束"** 双向控制模式。
- 反向提示词通过CLIP模型的attention机制抑制特定特征，产生像素级污染过滤
- **正负向权重比：1:1 至 1:2**
- 负向权重等效于正向的 0.6~0.8 倍抑制强度

### 通用负面词基础模板（必加）
```
blurry, deformed hands, extra fingers, text, watermark, logo, low quality
```
- 全部使用**英文小写逗号分隔**
- 禁止混用中文标点或全角符号
- 禁止同时输入中文和英文同义词（如同时"模糊"和"blurry"）

### 场景化负面词配置
- **人像类**：畸形五官、皮肤瑕疵、模糊面部、错误头发、多余手指、融合手指、五官不对称、闭眼异常、表情僵硬
- **产品类**：模糊纹理、色彩失真、产品变形、比例失调、材质失真、划痕瑕疵、背景杂乱、反光异常
- **视频类**：闪烁跳帧、穿模变形、画面抖动、帧丢失、前后帧不一致、主体瞬移、运镜混乱

### 正负分离铁则
- 正向提示词：**只描述"我要什么"**
- 负面提示词：**只描述"我不要什么"**
- 严禁正负描述混写

### 权重控制语法
- 双括号增强：`((blue-eyed cat))` → 权重 ×1.2
- 单括号减弱：`(soft sunlight)` → 权重 ×0.8
- 核心主体权重：1.5~1.7
- 细节特征权重：1.4
- 环境场景权重：≤1.2
- **负面词不加权重**

## 二、具象化替代抽象化

### 抽象词汇替换原则
AI理解具体客观的视觉元素，而非抽象的诗意或情感。

**错误示范**：一首关于乡愁的现代诗 / 高级感 / 氛围感

**正确做法**：描述具体的物体、颜色、材质、光线和构图

### 四维度拆解法
所有描述必须拆解为 **"材质 + 光影 + 形态 + 空间"** 四个可量化维度。

**示例**：
- ❌ "好看的猫" → ✅ "橘色短毛猫，绿眼睛，趴在木质书架上，爪子搭着一本翻开的童话书"

### 三层结构提示词构建法
1. **第一层：核心主体**（名词短语锁定对象）→ "柴犬幼犬"
2. **第二层：环境锚点**（必须含空间动词）→ "蜷缩在旧木箱角落" / "跃起扑向空中飞盘"
3. **第三层：表现基调**（摄影参数或艺术流派）→ "f/1.4 大光圈浅景深" / "宫崎骏动画风格"

### 场景变量格式
场景变量**仅置于提示词末尾**，格式统一：
```
……，[场景]，[光照]，[构图]
```

## 三、风格词精准化

### 细分风格描述体系
避免"卡通""好看"等模糊词，使用细分风格：
- **美术绘画风格**：古风插画、新中式国风、水彩手绘、厚涂油画、赛璐璐动漫、扁平插画、像素风、极简主义、线稿手绘
- **摄影写真风格**：日系清新、韩系氛围感、复古胶片、赛博朋克、蒸汽波、国潮风、宫崎骏动画风
- **中英文结合加分**：`治愈扁平风格 Healing flat style`

### 主辅风格权重标注法
- 主风格：直接书写，不加括号 → `赛博朋克城市夜景`
- 辅风格：双括号标注数值 → `((水墨晕染：0.4))`
- 数值范围：**0.3–0.6**（防止压制主风格）
- **单条提示词最多 2-3 个核心风格词**

### 禁止出现
- 英文介词 "and""with"（干扰权重解析）
- 无意义词汇"好看的""精美的"

## 四、系列内容一致性控制

### ID-LoRA 技术
通过低秩适配方式注入目标人物面部几何参数，锁定眼距、鼻梁高度、下颌线弧度等关键指标。

**标准语法**：
```
(lora: 角色名_v1:0.75)
```
- **实测最优权重：0.75**（保留自然微表情与强制脸型一致的平衡点）
- 过高质量疑：表情僵硬、动作失真
- 过低：无法抑制主模型随机性

### 随机种子固定策略
1. 生成第一段视频后，打开"生成详情页"
2. 找到"随机种子/S/Seed"一串数字，复制
3. 后续同系列视频，在"高级设置"中粘贴此种子值
4. **种子号只固定画风和人物，不固定动作和剧情**

### 多重约束保障机制
- **镜头切换防变脸**：分辨率全程固定 1080P；风格模型全程统一"视频3.0"
- **参考图校准**：勾选"风格模仿"而非"内容重构"，使用首尾帧+参考图三重校准

### 最佳实践参数
- 风格强度（style strength）：**60-80**（过高易失真）
- 参考图模式：**风格模仿**

## 五、功能绑定优化

### 首尾帧引导功能
上传首帧图和尾帧图，AI自动生成平滑过渡动画。

**操作流程**：
1. 选择"首尾帧"模式，准备风格统一的两张图片
2. 上传（建议分辨率一致，优先 1024×1024）
3. 输入过渡描述："猫咪逐渐变形成咖啡杯，变形过程流畅自然，背景保持不变"
4. 设置：视频长度10/15/20秒；过渡强度70%

### 口型同步功能
**操作流程**：
1. 导入角色视频/图像素材
2. 上传配音文件（中文或英文）
3. 开启"自动唇形同步"开关

**手动微调**：定位不准确时间点，以每5帧为单位微调口型

### 运镜控制与提示词融合
```
[运镜类型]，[速度控制]，[主体描述]，[场景]，[构图]
```

**运镜类型**：推镜头、拉镜头、摇镜头、旋转运镜、稳定跟拍

**进阶运镜**：希区柯克变焦、甩镜、一镜到底、过肩镜头、第一人称视角、第三人称跟随运镜

### 标准化调用模板
- **首尾帧模板**：`首尾帧：从花苞闭合到完全绽放，运镜：缓慢推近`
- **局部重绘模板**：`局部重绘：人物面部表情从严肃改为微笑，保持其他部分不变`
- **口型同步模板**：`口型同步：配音文件.mp3，要求口型与台词完全同步，表情自然连贯`
- **多镜头组合模板**：
  - 镜头1：全景，缓慢推镜，古风庭院内，身着白色汉服的女生坐在石桌前，抬手煮茶
  - 镜头2：近景，平稳横移，特写女生纤细的手拿起茶杯，指尖轻轻拂过杯沿
