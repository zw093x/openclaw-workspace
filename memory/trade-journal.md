## 模板

每次操作按以下格式记录：

```
### YYYY-MM-DD HH:MM | [买入/卖出] [股票名] [数量]股 @ [价格]

**决策背景：**
- 当时看到什么信息/信号
- 大盘环境如何
- 板块/个股处于什么位置

**决策逻辑：**
- 为什么认为应该操作
- 预期目标是什么
- 计划的止损/止盈位

**情绪状态：**
- 冷静/焦虑/贪婪/恐惧/冲动
- 是否受到他人影响
- 是否急于操作

**事后结果：**
- 实际盈亏
- 操作后走势是否符合预期
- 偏差原因分析

**经验总结：**
- 这次操作做对了什么
- 做错了什么
- 下次类似情况应如何处理
```
