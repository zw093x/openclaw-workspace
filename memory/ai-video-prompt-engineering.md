# AI视频生成提示词工程 v2.0（融合版）

> 整合三大平台：ComfyUI + 即梦(JiMeng) + Seedance
> 更新日期：2026-04-15

---

## 一、三平台定位对比

| 平台 | 定位 | 优势 | 限制 |
|------|------|------|------|
| **ComfyUI** | 本地工作流 | 最高控制力，可自定义节点 | 需要硬件配置 |
| **即梦(JiMeng)** | 字节出品 | 中文友好，图片生成强 | 视频时长有限 |
| **Seedance 2.0** | 分镜专家 | 专业分镜模板，运镜精准 | 不支持写实真人脸 |

---

## 二、Seedance 分镜提示词公式（新增）

### 核心公式
**【风格】+【主体描述】+【场景环境】+【动作行为】+【镜头语言】+【光影氛围】+【约束词】**

### 时间轴规范（每个片段必须包含）
1. **景别**：远景/全景/中景/近景/特写
2. **运镜/转场**：推/拉/摇/移/硬切/渐变/叠化
3. **画面描述**：主体运动 + 环境运动 + 背景运动 + 特效运动
4. **声音**：配乐/音效/对白

### 素材引用语法
```
@图片 1 作为首帧
@视频 1 参考运镜
@音频 1 用于配乐
```

## 三、镜头语言速查表

### 景别
| 景别 | 说明 | 适用场景 |
|------|------|---------|
| 远景 | 广阔空间，环境为主 | 开场建立场景 |
| 全景 | 人物全身，环境完整 | 展示人物与环境关系 |
| 中景 | 膝盖以上 | 日常对话、动作展示 |
| 近景 | 胸部以上 | 表情变化、情绪传递 |
| 特写 | 脸部/局部细节 | 强调情绪、关键物品 |

### 运镜
| 运镜 | 说明 | 效果 |
|------|------|------|
| 推镜头 | 摄像机向前推进 | 强调主体 |
| 拉镜头 | 摄像机向后拉远 | 展示环境 |
| 环绕 | 360度旋转 | 炫酷、展示主体 |
| 硬切 | 直接切换 | 节奏紧张 |
| 渐变 | 淡入淡出 | 情绪过渡 |

## 四、画面元素运动描述规范

**每个时间段必须描述四类元素：**

| 元素类别 | 描述内容 | 动词 |
|---------|---------|------|
| 主体运动 | 角色的动作、表情变化 | 迈步、抬头、滑落、抽搐 |
| 环境运动 | 天空，云，光影、风 | 飘动、流动、扩散、摇晃 |
| 背景运动 | 人群、物体、粒子 | 奔跑、飘散、坠落、飞溅 |
| 特效运动 | 火花、雨滴、烟雾 | 飞溅、喷射、扩散、飘散 |

**动词选择指南：**
- 轻小物体（树叶/纸片/尘埃）→ 飘动、飘落、飘散
- 光影/烟雾/液体 → 移动、流动、扩散
- 固体物体 → 震碎、坠落、掉落
- 植物/旗帜 → 摇曳、摇晃、摆动

## 五、主体描述扩展库（适用于所有平台）

### 人物描述五维度
| 维度 | 内容 | 示例 |
|------|------|------|
| 身份 | 职业或角色 | 东方女剑客、赛博朋克黑客、职场白领 |
| 外貌 | 年龄+发型+五官 | 25岁、高马尾、丹凤眼、冷白皮 |
| 服饰 | 材质+颜色+款式 | 丝绸旗袍、黑色皮质战甲、oversize卫衣 |
| 表情 | 情绪形容词 | 神情冷峻、眼神坚定、略带忧郁 |
| 稳定性约束 | 保障画面效果 | 面部特征保持一致、不扭曲 |

### 外貌词汇扩展
- **脸型**：国字脸、鹅蛋脸、圆脸、瘦长脸
- **眼睛**：丹凤眼、杏仁眼、深邃眼眸、炯炯有神
- **鼻子**：高鼻梁、挺拔的鼻梁
- **嘴巴**：薄唇、樱桃小嘴、嘴角微翘
- **眉毛**：剑眉、柳叶眉、眉峰高挑
- **肤色**：冷白皮、小麦色、古铜色
- **发型**：高马尾、丸子头、长发披肩、短发

## 六、敏感词合规替换

### 通用高风险词
| 高风险词 | 合规替换 |
|---------|---------|
| 真实、真人、实拍 | 写实风格、真人质感、纪实感 |

| 钞票、现金 | 货币、资金、财富 |

## 七、与即梦提示词工程的融合

### 即梦精准词库核心
- 主体特征 + 核心动作 + 运镜 + 时长 + 画面比例
- 详见 `jimeng-precision-library.md`

### Seedance vs 即梦 场景选择
| 场景 | 推荐平台 |
|------|---------|
| 角色全身+运镜 | Seedance（分镜模板更全） |
| 角色表情特写 | 即梦（图片生成更强） |
| 复杂工作流控制 | ComfyUI（本地节点控制） |
| 快速原型验证 | 即梦（响应快） |
| 电影级分镜 | Seedance（模板最专业） |
