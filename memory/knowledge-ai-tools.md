**版本**：v1.0
**适用平台**：Stable Diffusion WebUI (A1111/Forge) / ComfyUI v0.3+
**飞书文档**：https://feishu.cn/docx/NkumdADrtoz5lcxEFR4cRXuqnJb

### 核心结论速览

| 结论 | 详情 |
|------|------|
| 框架优先级 | 主体(35%) > 风格(20%) > 光影(18%) > 构图(12%) > 场景(10%) > 画质(3%) |
| 权重安全上限 | SD1.5上限1.4，SDXL/Flux上限1.3 |
| 位置法则 | 核心主体前置（前1/3），风格词中段，画质词末尾 |
| 权重推荐区间 | 1.2~1.4为安全区，>1.5开始有形态扭曲风险 |

### 模块A（CLIP+权重）
- 前20个token注意力最强，核心主体词应置于前1/3
- BPE分词影响："cyberpunk"≠"cyber"+"punk"，单独写更稳定
- 权重边界：0.5~0.8适合背景，0.9~1.1为基准，**1.2~1.4推荐**，1.5+有副作用

### 模块B（风格三要素）
- 艺术家权重：角色立绘1.2~1.4，概念艺术流派0.8~1.0
- 冲突避免：水彩+厚涂、赛璐璐+油画、摄影+动漫同时出现

### 模块C（光影构图）
- 伦勃朗光=戏剧感(45°侧光+三角光斑)，蝴蝶光=复古美妆感
- 影调体系：高调/低调/硬调/软调/中间调各有固定心理效应

### 模块D（LoRA+ControlNet）
- 角色LoRA：CLIP权重0.8~1.0；风格LoRA：UNet权重0.7~1.0
- 提示词主风格，ControlNet主结构
- 主体一致性技术：IP-Adapter > Reference-Only，InstantID专精人脸

### 补充说明
本报告基于子任务深度研究输出，包含完整底层原理、权重边界数据、ComfyUI配置示例及APE自动化优化流程。
