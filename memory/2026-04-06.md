### AI绘画/视频提示词研究（豆包课程，PDF已存）
**文件：** `/root/.openclaw/media/inbound/å_è_å_ç_å_¹è_0406---3317de0b-13f4-400f-a018-3e9311308c7f.pdf`
**核心认知：**
- 提示词本质=给模型提供视觉锚点+约束边界+对齐语义范式，而非单纯场景描述
- 不同模型提示词逻辑完全不同，必须先选定1个模型深耕
**模型提示词体系对比：**
| 模型 | 代表 | 提示词逻辑 |
|------|------|-----------|
| 闭源创意 | Midjourney/DALL-E 3 | 自然语言长句优先 |
| 开源可控 | SD/SDXL/Flux | 结构化关键词+权重，精确控制 |
| 长视频 | Sora | 电影级叙事文本，多镜头 |
| 短视频 | Pika/Runway Gen-3 | 精准动作+镜头描述 |
| 开源视频 | SVD/AnimateDiff | 兼容SD体系+运动模块 |
**AI绘画万能提示词框架（权重优先级）：**
```
核心主体（带核心特征:1.3） + 场景环境 + 构图视角 + 光影色调 + 视觉风格 + 画质标准 + 约束规则
```
示例：`[橘色布偶猫:1.3]，趴在木质书桌上，旁边有翻开的书和热拿铁，中景平视，柔和侧逆光，日系治愈，吉卜力画风，8K`
**SD权重语法：**
- `()` 提升权重，`[]` 降低权重
- `(关键词:1.2)` 精准权重赋值
- 核心主体放最前端，前置权重>后置权重
**风格锁定三要素：**
1. 艺术家/IP风格（宫崎骏/莫奈/新海诚）
2. 艺术流派（印象派/赛博朋克/国潮水墨）
3. 媒介材质（油画/水彩/C4D渲染/胶片）
**可控生成配套技术：**
- 模型/LoRA适配：触发词激活模型能力
