# 盘中异常主动深挖机制

*最后更新: 2026-03-26*
*用途: 发现异常信号时，自动查原因后给出结论，不把问题丢给P工*

---

## 触发条件

当盘中监控检测到以下任一信号时，自动启动深挖：

| 信号 | 阈值 |
|------|------|
| 涨跌幅异动 | 单只股票盘中波动超±2% |
| 量能异常 | 成交量超5日均量1.5倍 |
| 技术突破 | 价格触及或突破MA5/10/20/支撑/压力位 |
| 板块联动 | 持仓三只同向大幅波动 |
| 大盘异动 | 上证波动超±1% |

## 深挖流程

```
发现异常信号
    │
    ├─→ 1. 查个股新闻（30秒内）
    │     ├── 同花顺/东方财富个股新闻
    │     ├── 公司公告
    │     └── 券商研报更新
    │
    ├─→ 2. 查板块联动（30秒内）
    │     ├── 同板块其他股票表现
    │     ├── 板块资金流向
    │     └── 行业政策/新闻
    │
    ├─→ 3. 查资金面（15秒内）
    │     ├── 北向资金实时
    │     ├── 主力资金流向
    │     └── 融资融券变化
    │
    ├─→ 4. 查外部因素（30秒内）
    │     ├── 大盘走势
    │     ├── 外盘（美股期货/亚太）
    │     ├── 汇率/大宗商品
    │     └── 突发新闻
    │
    └─→ 5. 综合判断（15秒内）
          ├── 原因定性（利好/利空/技术面/资金面/无基本面）
          ├── 持续性判断（一次性/可能持续）
          ├── 操作建议（持有/加仓/减仓/观望）
          └── 推送给P工
```

## 输出格式

深挖完成后，以以下格式推送：

```
📊 [股票名] 盘中异动分析

异动：[涨跌幅] + [量能变化]

🔍 原因排查：
- 个股新闻：[有/无] → [具体内容]
- 板块联动：[有/无] → [板块表现]
- 资金面：[北向/主力动向]
- 外部因素：[大盘/外盘/汇率]

📋 判断：[利好/利空/技术面/资金面/噪音]
🎯 建议：[具体操作建议]
⚠️ 风险提示：[需要注意什么]
```

## 示例

```
📊 中国动力 盘中异动分析

异动：+3.20% 放量1.8x（触及MA20压力位32.72）

🔍 原因排查：
- 个股新闻：有 → 中船系获LNG船大单，中国动力为船用发动机供应商
- 板块联动：有 → 中国船舶+1.5%，中船防务+2.1%，板块共振
- 资金面：北向今日净流入船舶板块
- 外部因素：大盘平稳，无外部冲击

📋 判断：行业利好驱动 + 板块共振，非纯技术波动
🎯 建议：短期偏多，关注32.72能否有效突破站稳。若站稳可看33.50
⚠️ 风险提示：MA20是关键压力位，若冲高回落则可能回踩MA10(31.24)
```

## 与盘中监控的集成

修改 `scripts/stock_realtime_monitor.py` 的预警逻辑：
- 原来：发现异常 → 直接推送
- 现在：发现异常 → 先深挖 → 带原因和判断一起推送

Cron任务指令中已包含深挖要求：
"如有预警信号，先分析原因（查新闻、板块联动、资金面），然后给出带原因的预警推送。"
