# Scene Memory 技能 — 场景化记忆系统 v1.1

> 基于 memory-tdai 场景记忆理念，为 OpenClaw 构建的轻量级场景化记忆技能。
> 
> **LCM 双层记忆架构：** Scene Memory（静态知识层）+ Lossless-Claw（动态对话层）协同工作。

## 双层记忆架构

```
┌─────────────────────────────────────────────────────┐
│  Scene Memory（静态知识层）                           │
│  ─────────────────────────────────────────────────  │
│  • 分场景结构化沉淀（投资/运维/AI/宝宝）              │
│  • 热度追踪，高频知识优先                             │
│  • 文件路径: skills/scene-memory/references/        │
│  • 关键词检索 → 快速定位                             │
└─────────────────────────────────────────────────────┘
                         ↕ 互补
┌─────────────────────────────────────────────────────┐
│  Lossless-Claw / LCM（动态对话层）                   │
│  ─────────────────────────────────────────────────  │
│  • SQLite + DAG，每条消息都保留                       │
│  • lcm_grep 语义搜索所有历史对话                      │
│  • lcm_expand 展开摘要还原原始内容                    │
│  • 数据库: /root/.openclaw/lcm.db                   │
│  • 状态: enabled=true, threshold=0.75               │
└─────────────────────────────────────────────────────┘
```

## LCM 工具使用（融合进 Scene Memory 体系）

### LCM 命令
```bash
/lcm              # 查看状态：版本/DB大小/summary数量
/lcm doctor       # 扫描损坏或截断的summary
```

### LCM 检索工具（重要！）
> 这些是 OpenClaw 内置工具，在对话中直接调用，不是 bash 命令。

**在对话中使用方式：**
- 语义搜索 → 使用 `lcm_grep` 工具（我可以直接调用）
- 展开摘要 → 使用 `lcm_expand` 工具（我可以直接调用）
- 查看摘要 → 使用 `lcm_describe` 工具（我可以直接调用）

**使用示例：**
```
用户: 查找我之前关于船舶的所有讨论
→ 我调用 lcm_grep("船舶") 进行语义搜索
→ 根据结果用 lcm_expand 展开感兴趣的对话
```

### LCM 使用时机
| 场景 | 工具 | 说明 |
|------|------|------|
| 查某段历史对话细节 | `lcm_grep` → `lcm_expand` | 语义搜索 + 展开摘要 |
| 回顾某次决策过程 | `lcm_grep` → `lcm_expand` | 还原完整对话链 |
| 查找特定时间的事件 | `lcm_grep` 时间/日期 | 跨会话搜索 |

## 核心能力

- **分场景存储**: 4大核心场景，结构化沉淀用户信息
- **热度追踪**: 被引用越多越重要，优先展示
- **关键词检索**: 快速定位目标场景内容
- **增量更新**: 从对话中自动提取关键内容写入对应场景
- **LCM 融合**: 静态知识层 + 动态对话层，双重保障

## 核心脚本

```bash
# 列出所有场景
python3 /root/.openclaw/workspace/skills/scene-memory/scene_memory.py list

# 搜索关键词（搜索所有场景）
python3 /root/.openclaw/workspace/skills/scene-memory/scene_memory.py search <关键词>

# 读取指定场景
python3 /root/.openclaw/workspace/skills/scene-memory/scene_memory.py read <场景名>

# 追加内容到场景
python3 /root/.openclaw/workspace/skills/scene-memory/scene_memory.py append <场景名> <内容>
```

## 使用流程（推荐）

### Step 1: 对话前 — 加载背景
```
读取: skills/scene-memory/references/02-投资管理与交易策略.md
→ 了解用户持仓、策略偏好、数据架构
```

### Step 2: 对话中 — 知识检索
```
# 场景知识（精确）
scene_memory.py search 船舶

# 历史对话（语义）
lcm_grep 船舶
```

### Step 3: 对话后 — 沉淀新知识
```
→ 投资策略更新 → 写入"02-投资管理与交易策略.md"
→ 工具配置变更 → 写入"01-数字环境与工作流.md"
→ 新学到的技能 → 写入"03-AI技术与产品研究.md"
```
