# 报告模板

## 每日日报模板

```markdown
# {行业}日报 ({日期})

> {一句话摘要}

## 要点速览
- [🔥热点] {最重要的1-2条}
- [📊数据] {关键数据/指标}
- [💡洞察] {值得关注的趋势}

## 详细资讯
### {分类1}
- **{标题}**: {摘要} [来源]({url})

### {分类2}
...

## 对持仓/工作的影响
- {影响分析}

---
*来源: {sources} | 生成时间: {timestamp}*
```

## 每周综合模板

```markdown
# 综合周报 ({起止日期})

## 本周概要
> {3-5句话总结本周关键事件}

## 分项回顾

### 📈 股票表现
| 股票 | 周初 | 周末 | 周涨跌 | 关键事件 |
|------|------|------|--------|---------|

### 🤖 AI 动态
- {要点1}
- {要点2}

### 🏠 家庭/生活
- {要点}

### 📋 下周预告
- {下周重要事件}

---
*生成时间: {timestamp}*
```

## 行业深度报告模板

```markdown
# {行业}深度报告 ({日期})

## 核心发现
> {一段话总结}

## 事件详情
### 事件1: {标题}
- **背景**: ...
- **内容**: ...
- **影响**: ...
- **来源**: [{source}]({url})

## 趋势分析
{分析内容}

## 行动建议
- [ ] {建议1}
- [ ] {建议2}
```
