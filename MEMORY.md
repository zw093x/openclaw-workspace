```
└──────────────────┬───────────────────────────────────┘
                   ↓
┌──────────────────────────────────────────────────────┐

│  接收错误事件 → 分析模式 → 生成预防策略                │

└──────────────────┬───────────────────────────────────┘
                   ↓
┌──────────────────────────────────────────────────────┐
│  统一知识中枢 (intel_hub.py)                         │
│  汇聚所有系统知识 → 共享给自愈系统                    │

                   ↓

     ↓
修复时检查是否有预防策略 → 有则优先执行预防策略
```
